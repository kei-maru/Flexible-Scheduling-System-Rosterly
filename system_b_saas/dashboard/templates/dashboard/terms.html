利用規約

本利用規約（以下「本規約」）は、Veludo System B（以下「本サービス」）の利用条件を定めるものです。ご利用にあたり、本規約に同意いただく必要があります。

1. 適用

本規約は、本サービスの利用に関する当社とユーザーとの間の一切の関係に適用されます。

2. アカウント

ユーザーは、正確かつ最新の情報に基づきアカウントを利用するものとし、第三者による不正使用の防止に努めるものとします。

3. 禁止事項

  • 法令または公序良俗に反する行為
  • 不正アクセス、脆弱性の悪用、スクレイピング等の迷惑行為
  • 他のユーザーまたは第三者の権利侵害
  • 本サービスの運営を妨害する行為

4. 収集する情報

当社は本サービスの提供・改善・安全対策のため、以下の情報を収集する場合があります。

  • アカウント情報（ユーザー名、メールアドレス、Discord ID 等）
  • 予約・シフトなどの業務データ
  • 利用ログ（ページ閲覧、クリック、操作対象、遷移元、操作時刻）
  • 端末情報（ブラウザ種別、OS、画面サイズ、IP、タイムゾーン、言語設定 等）
  • Cookie 等の識別子

5. 利用目的

  • 本サービスの提供および運営
  • 不正利用の検知・防止、セキュリティ強化
  • 品質向上、機能改善、ユーザー体験の最適化
  • サポート対応、重要なお知らせの送信

6. 第三者提供

当社は、法令に基づく場合を除き、ユーザーの同意なく個人情報を第三者に提供しません。なお、Discord 等の外部サービスによる認証を利用する場合があります。

7. 免責

当社は、本サービスの利用により発生した損害について、法令に定める場合を除き一切の責任を負いません。

8. 規約の変更

当社は必要に応じて本規約を変更することがあります。重要な変更がある場合は適切な方法で通知します。

9. 準拠法・裁判管轄

本規約は日本法に準拠し、本サービスに関する紛争は当社所在地を管轄する裁判所を第一審の専属的合意管轄裁判所とします。
